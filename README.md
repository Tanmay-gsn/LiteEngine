LiteEngineA custom, data-driven 2D game engine built in C++. LiteEngine is designed with a modular architecture, featuring an abstracted rendering pipeline, a blocking event dispatcher, and centralized resource management. To rigorously validate the engine's subsystems, a fully playable "Breakout" clone is integrated natively.ArchitectureCore & LayerStack: Implements a scalable layer system to strictly order game logic and UI updates.Event Dispatcher: A blocking event routing system that intercepts OS inputs, eliminating input bleeding between layers.Abstracted Rendering: A platform-agnostic Renderer2D API currently backed by an OpenGL implementation utilizing GLAD and GLM.Resource Management: A centralized cache for shaders and textures utilizing hash maps to prevent redundant allocations.Developer Tooling: Native ImGui integration with viewport docking for real-time debugging, alongside comprehensive spdlog system logging.Performance & MetricsLiteEngine is architected for low overhead and continuous high-throughput execution. To validate the engine's core subsystems, strict stress testing was performed simulating a massive grid of 750+ active entities.Rendering Throughput: Maintained a stable 120 FPS using the custom batch rendering pipeline.Event & Logic Latency: CPU processing time from hardware interrupt to complete LayerStack resolution averaged < 1.0 ms per frame.VRAM Optimization: The centralized ResourceManager successfully prevented redundant texture allocations, resulting in a 99.8% memory savings (preventing exactly 787 MB of unnecessary GPU overhead).BenchmarksDependenciesOpenGLGLFWGLADGLMImGuispdlogBuild Instructionsgit clone --recursive [https://github.com/Tanmay-gsn/LiteEngine.git](https://github.com/Tanmay-gsn/LiteEngine.git)
